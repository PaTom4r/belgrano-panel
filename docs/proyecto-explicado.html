La oportunidad que descubrimos

Con Belgrano
+$4.987.500

CLC deja de pagar y empieza a GANAR plata. Incluso en el peor escenario, gana casi $5M/mes.

70 pantallas Samsung repartidas por toda la clinica — salas de espera, pasillos, urgencias, cafeteria.

6.000 personas al dia pasan frente a esas pantallas, sentadas, esperando, mirando.

MagicInfo es el software de Samsung que controla lo que se muestra en cada tele (como un control remoto gigante).

5 Que se Construyo

Los dos pilares del proyecto: documentos de negocio + plataforma web

Parte A: Documentos de Negocio

Todo lo que necesitas para vender el servicio y operar. Carpeta docs/

Modelo de Ingresos

Cuanta plata entra segun 3 escenarios de fill rate. Incluye CPM, revenue share, y proyecciones anuales.

Media Kit

Catalogo profesional para mostrar a anunciantes. Datos de audiencia, formatos, precios, paquetes.

Pipeline de Anunciantes

20+ empresas organizadas por categoria con pitch, presupuesto estimado y approach.

Politica de Contenido

Que se puede y que no mostrar en la clinica. Regulacion chilena (ISP, Ley 20.606).

Checklist de Auditoria

Lista de verificacion para la visita a CLC: licencias, servidor, red, seguridad.

Catalogo de Pantallas

Planilla para documentar las 70 teles: modelo, ubicacion, estado, zona.

Plan de Transicion

Paso a paso para cambiar de proveedor sin que se apague ninguna tele.

Guia de Seguridad

Como parchear la vulnerabilidad critica CVE-2024-7399 del servidor MagicInfo.

Parte B: Belgrano Panel (aplicacion web)

Tu centro de control. Se abre en el navegador. Carpeta belgrano-panel/

Dashboard

Resumen general: pantallas online, campanas activas, plata del mes

Screens

Las 70 pantallas con estado (verde = online), zona, modelo. Reinicio remoto.

Content

Subir videos e imagenes, crear playlists, programar horarios.

Campaigns

Campanas publicitarias: anunciante, fechas, zonas, frecuencia.

Advertisers

CRM de anunciantes: contacto, contrato, categoria, campanas.

Reports

Proof-of-play, graficos, revenue share para CLC, exportar CSV/PDF.

Para verlo en accion:

cd belgrano-panel
npm run dev

Abre http://localhost:3000 en tu navegador. Funciona con datos de demo — no necesita base de datos ni MagicInfo real.

6 Riesgos a Tener en Cuenta

Cosas que pueden salir mal y como evitarlas

⚠
Las licencias de MagicInfo pueden ser del proveedor actual Sin licencias no controlas las teles. Verificar ANTES de firmar con CLC.
⚠
Vulnerabilidad critica en el servidor (CVE-2024-7399) Un hacker podria tomar control. Se creo una guia de parcheo. Hacer antes de operar.
⚠
Las teles se pueden apagar durante la transicion Pantallas negras en una clinica = desastre. El plan de transicion incluye 2-4 semanas de operacion paralela.
⚠
No conseguir anunciantes rapido Sin ingresos no hay negocio. Por eso los primeros targets son ISAPREs que YA estan en CLC.
